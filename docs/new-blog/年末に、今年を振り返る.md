# 年末に、今年を振り返る

お疲れ様です。xxxx です。

気づけばもう年末ですね。
この時期になると、なんだかソワソワしてしますね。

「あれもやらなきゃ、これもまだ終わってない」と、
やり残したことのリストに目が行きがちです。
みなさんはいかがでしょうか。

もちろん、計画的に一年を締めくくるのは素晴らしいことです。
でも、100 点満点の振り返りなんて、そもそも必要なのでしょうか。

たくさんの目標を立てるのも素晴らしいですが、
あれもこれもと手を出すと、
結局どれも中途半端になってしまうこともありますよね。

それよりも、今年一番心に残った「できたこと」を一つだけ思い出してみる。
あるいは、来年「これだけは続けたい」と思うことを一つだけ見つけてみる。

それだけで十分なのかもしれません。

一つの小さな成功体験や、続けられそうな一つの習慣のほうが、
来年の自分をそっと支えてくれるお守りになる気がします。

完璧な年末なんて、きっとない。
だからこそ、今年一年がんばった自分を少しだけ認めて、
小さな「できた」を一つポケットに入れて新しい年を迎えられたら、
素敵だと思いませんか。

来年も、きっと良い年になりますように。
