# 小さな達成感

お疲れ様です。xxxx です。

大きな目標を立てるのは素晴らしいことですが、
達成には時間も労力もかかるので、そればかり追いかけていると
「まだまだゴールは遠いな…」と感じてモチベーションが
なかなか上がらないこともあるかと思います。

そんな時は、どんなに些細なことでも「よし、終わったぞ」という
小さな達成感を一つひとつ重ねていくと気持ちも前向きになるのでは
ないかと思います。

例えば、
「溜まっていたメールを全部返信した」
「デスク周りを整理整頓できた」
「読みかけだった本を１章読み終えた」

そんなことで良いのだと思います。

一日の終わりに、今日達成できたことを一つだけ思い出してみる。

それだけで「自分も意外と頑張っているな」と
少しだけ自分を認められる気がしませんか。

その小さな達成感が、次の「もうちょっとだけやってみよう」
という気持ちにつながっていく。

大きな目標に向かうための大切なエネルギーになるはずです。

一つひとつの「できた」を集めていくことで、
きっと大きな自信につながっていく。

そんな風に進んでいけると良いですね。
